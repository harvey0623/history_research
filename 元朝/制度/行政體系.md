# 元朝行政制度：一省制與行省體系研究

## 1. 歷史背景

元朝（1271年－1368年）的行政制度在中國歷史上具有獨特的地位，它既承襲了唐宋以來的漢法，又保留了蒙古草原的民族特色，形成了高度集權且帶有軍事殖民色彩的政治體系。這套制度的設計初衷在於確保蒙古貴族的絕對統治權，同時有效管理橫跨歐亞的廣大疆域。

## 2. 中央行政機構：一省制

與唐宋傳統的「三省六部制」不同，元朝精簡了中央機構，實行**一省制**。

- **中書省**：廢除門下省與尚書省，僅保留中書省為全國最高行政機構。中書省領有六部（吏、戶、禮、兵、刑、工），長官為中書令（通常由皇太子兼任），實際政務由左右丞相與平章政事負責。
- **樞密院**：負責全國軍事事務，體現了元朝「文武分治」但軍事優先的特徵。
- **御史台**：最高監察機構，負責監督官吏與行政效率，被稱為「天子耳目」。
- **宣政院**：元朝特有的機構，專門管理全國佛教事務以及**吐蕃（西藏）地區**的軍政事務，標誌著西藏正式納入中央行政管轄。

## 3. 地方行政制度：行省制度

元朝最重大的制度創新是[行省制](./行省制.md)（全稱「行中書省」）。這原是中央中書省派出的臨時機構，後來演變為常設的最高地方行政單位。

- **行省的劃分**：全國除「腹裏」（由中書省直轄的山東、山西、河北）外，設有十個行省。
- **權力制衡**：行省長官擁有極大的行政、兵權和財權，但必須受中央垂直領導，且實行「圓署會議制」（集體辦公），防止地方割據。
- **犬牙交錯**：採取「犬牙相入」原則劃分省界，打破自然地理界限（如秦嶺、淮河），使地方無險可守，從而加強中央集權。

## 4. 基層管理與社會結構

元朝在基層行政上採取了嚴格的族群管理與軍事化管理，與[四等制度](./四等制度.md)緊密結合。

- **行政層級**：行省之下依序設立**路、府、州、縣**。
- **達魯花赤**：各級地方政府中必須設置「達魯花赤」（蒙古語，意為鎮守者或掌印者），通常由蒙古人擔任，掌握最後決定權，確保蒙古貴族對地方的絕對掌控。
- **諸王投下**：保留了蒙古封建特色，部分土地和人口分封給宗室勳貴，形成某種程度上的「國中之國」，與郡縣制並存。
- **嚴密監控**：基層實施[里甲制](./里甲制.md)，透過蒙古甲主對漢人與南人進行二十戶一組的嚴密監視。

## 5. 制度特點與歷史影響

### 制度特點總結

| 特點         | 說明                                                 |
| ------------ | ---------------------------------------------------- |
| **高度集權** | 中書省總攬行政，行省制加強了對邊疆的控制。           |
| **二元體制** | 漢族官僚體系與蒙古「達魯花赤」、投下制度並行。       |
| **軍事色彩** | 樞密院權力極大，地方官員常帶軍職。                   |
| **族群等級** | 職位任用嚴格參考「四等人制」，形成制度化的族群歧視。 |

### 歷史意義

元朝的行省制度對後世影響深遠，奠定了現代中國省級行政區劃的基礎。然而，其制度設計中人為製造的階級壁壘（如四等人制與高壓里甲制），最終成為激化民族矛盾、導致元朝覆滅的核心原因。

## 研究結論

元朝的行政制度是一場規模宏大的政治實驗，成功解決了自唐末以來地方割據的積弊，強化了國家對遼闊邊疆（尤其是西藏、雲南）的實質管轄。然而，這種建立在民族不平等與軍事高壓基礎上的統治體系，雖具備極高的行政效能，卻缺乏社會認同的根基，最終在元末民變中迅速崩潰。
