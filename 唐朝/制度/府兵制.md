# 府兵制

## 歷史背景

府兵制起源於北朝西元 534 年（西魏大統元年）至 581 年（北周大象三年）期間，並於 618 年（唐武德元年）至 741 年（唐開元二十九年）發展至成熟鼎盛。此制度是初唐國力強盛的軍事基礎，核心概念為「兵農合一」。

## 核心內容摘要

### 1. 制度設計與運作

- **編制單位**：全國設立「折衝府」（軍府），士兵編入府籍，稱為「府兵」。
- **兵農合一**：士兵平時務農，戰時當兵。國家依據「均田制」配給土地，士兵需自備部分武器與糧餉。
- **輪番服役（番上）**：府兵需定期前往京師（長安）衛戍或派駐邊疆，這種輪值制度確保了中央對軍權的直接掌控。

### 2. 兵源與身份性質（世襲性分析）

- **初期（西魏、北周）**：具備強烈**世襲色彩**。當時實行「軍民分治」，府兵擁有專門的軍籍，身份世代相傳，形成了初期的軍功貴族集團。
- **鼎盛期（隋唐）**：轉為**非世襲的「點選制」**。
  - **選拔機制**：政府根據「均田制」下的家庭財力（資產）、丁口數與體格進行挑選。通常從中等以上農戶中選拔。
  - **退役機制**：士兵在服役期限屆滿（通常為 60 歲）或因傷病即可解役，恢復普通平民（農民）身份，不具備永久世襲義務。
- **本質區別**：唐代府兵是「兵農合一」的義務役（基礎是土地）；明代衛所制則是「軍戶世襲」的職業役（基礎是身份）。

### 3. 制度優點（初唐強盛原因）

- **財政負擔極低**：國家無需支付常備軍軍餉，節省大量行政與軍事開支。
- **抑制軍閥割據**：軍權集中於中央，士兵平時分散於各地務農，地方將領難以長期私養兵力。
- **戰鬥力穩定**：府兵多為具備經濟能力的自耕農（均田戶），體質較佳且具備一定的軍事訓練。

### 4. 致命問題與結構性矛盾

- **均田制崩壞（關鍵原因）**：隨著土地兼併嚴重，農民失去土地，府兵失去賴以生存的經濟基礎，導致制度瓦解。
- **戰爭性質改變**：武則天至唐玄宗時期（西元 690 年－741 年），對外戰爭頻繁且時間拉長，府兵長期戍邊導致荒廢農務，負擔過重引發逃亡。
- **社會不公**：富人常透過手段逃避兵役，負擔完全落在貧農身上，加速了基層社會的崩潰。

## 軍制演化與跨代比較

| 面向         | 府兵制 (唐初)        | 募兵制(唐中後期)      | 衛所制(明朝)           |
| :----------- | :------------------- | :-------------------- | :--------------------- |
| **核心模式** | 兵農合一             | 職業軍人              | 軍戶世襲 (半兵農)      |
| **經濟基礎** | 靠土地養兵 (均田制)  | 靠金錢養兵 (國庫)     | 靠身份養兵 (軍戶制)    |
| **國家負擔** | 極低                 | 極高                  | 中等 (半自給)          |
| **戰力評價** | 中等，適合作戰與屯墾 | 極強，專業化程度高    | 初期強，後期因腐敗弱化 |
| **致命缺陷** | 土地流失即瓦解       | 引發安史之亂 (軍閥化) | 軍戶逃亡、戰力低落     |

## 歷史影響與意義

府兵制的瓦解是唐朝歷史的重要轉折點。西元 742 年（唐天寶元年）後，唐朝全面轉向「募兵制」，雖然提升了專業戰力，卻也導致軍權下放至地方節度使，最終釀成安史之亂

這條演化線展現了中國歷代政權在「成本、戰力、控制力」三者間的無解三角困境：

1.  **府兵制**：追求低成本與高控制，但毀於經濟結構（土地）改變。
2.  **募兵制**：追求高戰力，但導致國庫空虛與政治失控。
3.  **衛所制**：試圖折衷，卻因世襲制度僵化而走向腐敗。

## 研究結論

府兵制不僅是一項軍事制度，更是建立在均田制上的社會經濟體系。其興衰見證了唐代從「小農自足」轉向「專業軍事官僚化」的必然過程。一旦國家無法保障士兵的土地所有權，這種「低成本國防」便難以為繼。
