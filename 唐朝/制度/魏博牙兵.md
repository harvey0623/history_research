# 魏博牙兵制度研究

## 1. 歷史背景與起源

魏博牙兵（又稱牙軍）是唐朝中晚期最強大的地方軍事利益集團，其根源可追溯至唐代大曆年間（西元766年－779年）。安史之亂後，魏博鎮首任節度使田承嗣為鞏固其割據勢力，佔領魏博六州後，從十萬徵發兵中精選萬人（亦有記載為五千人）組成親衛親軍，駐紮於節度使治所的「牙城」內，故稱「牙兵」。

## 2. 制度特徵與核心內容

### 2.1 精銳化與駐紮核心

- 萬中選一：牙兵是從十萬大軍中挑選出的最精銳力量，戰鬥力極強。
- 地位尊崇：其地位類似於唐朝中央的神策軍，享有優於一般軍隊的糧餉待遇，且直接護衛節度使權力核心。

### 2.2 地方軍閥化的結構

- 父子世襲：牙兵實行世襲制，且全由魏博本地人組成。
- 血緣聯姻：長期以來，牙兵集團內部透過聯姻交織成龐大的利益網路。
- 僱傭性質：牙兵轉向純粹的職業僱傭兵，缺乏對國家的忠誠，其效忠對象僅為能提供高額賞賜的主帥。

### 2.3 「長安天子，魏府牙軍」

這句當時流傳的民諺，生動描述了魏博牙兵的權勢已能與大唐皇帝分庭抗禮。到了魏博中後期，牙兵集團掌握了節度使的廢立大權，主帥若削減待遇或不合其意，動輒發生兵變（譁變）殺害主帥另立新君。

## 3. 魏博牙兵的覆滅過程

### 3.1 第一階段：羅紹威與朱溫的聯合屠殺（唐末）

- 誘因：西元905年，魏博節度使羅紹威因畏懼牙兵強權，決意剷除。
- 行動：西元906年，羅氏聯合宣武軍節度使朱溫，利用喪事名義引兵入城，並預先破壞軍械。
- 結果：聯軍突襲牙兵駐地，誅殺全體牙兵並屠戮家屬八千餘戶。羅紹威雖除心腹大患，卻導致魏博元氣大傷，從此喪失藩鎮獨立性，感嘆「鑄成大錯」。

### 3.2 第二階段：後唐明宗的最後鎮壓（五代時期）

- 銀槍效節都的建立：西元912年，後梁名將楊師厚接管天雄軍（魏博），為鞏固勢力，選拔數千精銳組建銀槍效節都。這支軍隊待遇極其優厚，衣食軍械皆由官府供給最好的，且在城中享有法外特權，實質上繼承了老牙兵傲慢、強悍且唯利是圖的基因，被稱為「第二代牙兵」。
- 鄴城兵變與皇權更迭：西元926年，駐守瓦橋關的魏博兵因超期服役引發「鄴城兵變」，推舉趙在禮佔領魏州。後唐莊宗李存勗派名將李嗣源前往平叛，結果李嗣源的部隊在城外譁變，被迫與城內銀槍效節都合流，擁立李嗣源造反。此事件導致莊宗在洛陽覆滅，李嗣源（後唐明宗）順勢登基。
- 盧台屠殺與清算（西元927年）：李嗣源登基後，深感這群「擁立有功」的士兵同樣能輕易將其廢黜。西元927年，他藉口調防北方遼國，將銀槍效節都誘至盧台（今河北境內），隨即派大將藥彥稠率兵將其數千人及其隨軍家屬全數坑殺。同時下令清算魏州城內的牙兵餘黨，徹底根除了威脅。至此，存在15年、左右五代初期政局的第二代魏博牙兵被徹底抹除。

### 3.3 結構性瓦解（後晉時期）

後晉高祖石敬瑭為解決魏博長期割據與將領（如范延光、楊光遠）勢力膨脹的問題，於西元938年採納大臣\*\*桑維翰的建議，將魏博鎮「一分為三」，從根源上瓦解其「天下雄藩」的根基：

- 削弱核心：將魏州升格為「廣晉府」（鄴都），使天雄軍節度使僅能轄制魏州一地。
- 拆分州郡：將原屬魏博的相州、洺州劃出，另設彰德軍；將貝州、博州劃出，另設永清軍；澶州則劃歸中央直轄。
- 調虎離山與分化：藉行政調整將強橫的將領楊光遠調離魏州，並將其麾下親兵大將分封至各州擔任刺史，兵不血刃地解除了武裝威脅。至此，存在百年的魏博割據土壤被徹底剷除。

## 4. 歷史影響與研究結論

魏博牙兵的興衰反映了唐末五代「兵驕將悍」的典型政治風氣。

1.  藩鎮割據的極致：其世襲與地緣性，使得藩鎮從軍事單位演變為獨立的政治實體。
2.  社會結構的破壞：這種唯利是圖的僱傭兵文化，導致政權頻繁更迭，造成長期動盪。
3.  鑄成大錯的典故：羅紹威「傾魏博之鐵不能鑄此錯」的感嘆，成為歷史上對「飲鴆止渴」式決策的深刻警戒。

魏博牙兵的消失，象徵著唐式藩鎮體系的崩潰，也為後續宋代「重文輕武」與「強幹弱枝」政策的形成提供了歷史教訓。
