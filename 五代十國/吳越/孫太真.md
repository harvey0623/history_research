# 孫太真

## 歷史背景

孫太真（？—976年，宋開寶九年），吳越國末代國王錢弘俶之妻，在吳越國內部被尊稱為「黃妃」。其生活於五代十國末期，正值中原大宋政權（趙匡胤）積極統一南方割據政權的關鍵時刻。作為偏安江南的吳越國王妃，孫太真在維繫家國平安與處理對宋外交關係中，扮演了極其關鍵且具備高度政治智慧的角色。

## 核心內容摘要

### 1. 身份與軍事聯姻背景

孫太真的婚姻本質上是吳越政權與地方強大軍事勢力的結合：

- **將門出身**：祖父孫承露曾任唐代睦州刺史，父親孫漢英為吳越核心將領。
- **軍事實力**：孫家掌握蘇杭地區最精銳的步騎武裝，是錢氏王權的重要軍事保險。
- **情感轉化**：雖始於政治聯姻，但孫太真與錢弘俶最終發展出相謬以沫的深厚感情。

### 2. 夫人外交與主權確認

在公元976年（宋開寶九年）朝見宋廷的過程中，孫太真展現了卓越的政治博弈技巧：

- **確立主權歸屬**：面對宋皇后的豐厚賞賜，孫太真以「此件官家物，等守之以待今日歸」一句話，從法理上確認趙匡胤為天下唯一合法主君，將吳越國定位為「代守者」而非對等政權。
- **去威脅化策略**：透過極致的卑微與隱忍，封殺了宋廷主戰派發兵吳越的藉口，為江南百姓爭取了和平過渡的空間。

### 3. 雷峰塔（黃妃塔）的興建動機

雷峰塔的興建（始於975年，宋開寶八年）承載了多重深層意涵：

- **表層動機**：最初是錢弘俶為重病的孫太真祈福續命之舉。
- **政治倒戈與宣洩**：錢弘俶不惜耗費國帑興建十三層木塔（原計畫），是其壓抑一生附庸身份後的最後一次主權行使與情感爆發。
- **尊嚴的補償**：這座塔成為錢弘俶對妻子多年來在政治外交中犧牲尊嚴的一種回報與紀念。

## 歷史影響與意義：女性命運的殘酷對照

透過孫太真與大宋宋皇后的對比，可見兩者在權力更迭中的不同選擇與結局：

| 比較維度         | 孫太真 (吳越王妃)                                    | 宋皇后 (大宋皇后)                                        |
| :--------------- | :--------------------------------------------------- | :------------------------------------------------------- |
| **權力博弈選擇** | **卑微中守住慈悲**：選擇徹底放棄自尊以換取家國太平。 | **頂峰中的權力爭奪**：在太祖駕崩後試圖搶先立儲以保權位。 |
| **政治判斷**     | 看清絕對暴力下的現實，採取「去威脅化」。             | 錯估形勢與人心（如王繼恩的背叛），導致滿盤皆輸。         |
| **最終命運**     | 雖因操勞於同年病逝，但換來江南免於戰火。             | 餘生19年處於趙光義的打壓與屈辱中，死後亦不得合葬與入廟。 |

## 研究結論

孫太真並非歷史中沈默的後宮附庸，而是一位具備深邃政治眼光的女性政治家。她在「夫人外交」中的卑微，實則是一種隱含「孤勇」的戰略選擇。她用個人尊嚴的折損，成功化解了吳越亡國前夕最猛烈的殺機。雷峰塔（黃妃塔）不僅是一座佛教建築，更是亂世中一段以卑微換取慈悲、以犧牲換取太平的歷史見證，其歷史定位應被視為江南和平過渡的重要推手。
