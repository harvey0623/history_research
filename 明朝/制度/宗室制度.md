# 明朝宗室制度

明朝宗室制度由開國皇帝朱元璋一手制定，旨在確保皇族子孫永保富貴並鎮守天下。然而，這套制度隨著時間推移，演變成嚴重的財政與社會問題，最終成為明朝滅亡的重要誘因。

## 1. 制度設計與運作特徵

- **設立目的**：確保子孫優渥生活，並以親王鎮守各地，防止功臣或外戚奪權。
- **世次降級分封制**：
  - 皇帝之子封為**親王**。
  - 親王嫡長子繼承為**郡王**，其餘子嗣依序降為**鎮國將軍**、**輔國將軍**、**奉國將軍**、**奉恩將軍**。
  ```
    親王
    ├─ 嫡長子 → 郡王
    │   ├─ 嫡長子 → 輔國將軍
    │   │   └─ 子 → 奉恩將軍
    │   └─ 其他 → 奉國將軍
    │       └─ 子 → 奉恩將軍
    └─ 其他子 → 鎮國將軍
        └─ 子 → 奉國將軍
            └─ 子 → 奉恩將軍
  ```

  - 第五代以後不再分封爵位，僅保留宗室身分。此制度本意是防止高爵位壟斷，但未能阻擋人口爆炸。
- **國家終身供養**：宗室免稅、免役，不受地方官府管轄，從出生、婚嫁到喪葬均由國家財政負擔，生活標準極高。
- **嚴格禁令與政治閹割**：為防止宗室造反，嚴禁其擔任官職、掌握軍權或從事商業。親王形同被軟禁於王府，且嚴禁各王私下見面（二王不相見）。
- **生育獎勵機制**：因被禁止工作，生育成為宗室增加收入的唯一合法途徑。每多生一名子嗣，國家即依等級增發俸祿。

## 2. 瘋狂生育與人口爆炸

- **按人頭計給**：每出生一子可多領約 1,000 石俸祿。
- **極致斂財**：王爺們為領取更多撥款而拼命納妾，甚至精確計算生理期以極大化生育效益。
- **人口增速**：宗室人口從明初的數十人，至明末推算已達近百萬人，增長速度約為全國平均的 10 倍。

## 3. 對國家與地方的毀滅性影響

### 財政危機

- **財政黑洞**：宗室俸祿支出呈幾何級數增長。明代中後期，許多省份（如山西、河南）的全省財政收入甚至不足以支付當地王爺的俸祿。
- **拖欠俸祿**：財政崩潰後，朝廷被迫長期拖欠俸祿，導致宗室內部貧富極端分化，底層宗室陷入貧困。

### 經濟壟斷與土地兼併

- **特權壟斷**：宗室利用影響力壟斷鹽業等暴利行業，並向皇帝索要自然資源（山林、礦產）。
- **瘋狂圈地**：各王府莊田動輒萬頃，侵占全國最肥沃的土地，導致農民失去生計。
- **稅權掠奪**：王爺常以「裝窮」為由，將地方商業收稅權據為己有，進一步削弱中央財政。

### 社會治安與法治破壞

- **地方黑勢力**：宗室擁有司法特權，地方官無權處置。部分宗室包庇盜賊、強搶民女，甚至親自帶人行劫。
- **凌辱官員**：當要錢、要糧的要求未被滿足時，宗室動輒毆打知縣甚至省級官員（布政使）。

## 4. 地方官員的困境

- **賦稅轉嫁**：官員為填補宗室俸祿黑洞，只能不斷向百姓「加派」賦稅，逼使農民賣房賣妻。
- **束手無策**：面對宗室的暴力與特權，地方官府完全沒有執法能力，只能任其作威作福。

## 5. 明末起義軍的血腥報復

由於兩百多年的積怨，明末農民起義軍（如李自成、張獻忠）對宗室實行了毀滅性的清算：

- **徹底屠殺**：採無差別對待，不論投降與否，凡朱姓宗室一律誅殺。
- **規模化清洗**：
  - **李自成**：在山西一地即殺害萬餘名宗室，大同代王府四千餘人於六日內被殺光。
  - **張獻忠**：喜好殘酷虐殺，如將楚王活活淹死、燒毀氣派的襄王府。
- **財產焚毀**：各地奢華的王府多被洗劫一空後付之一炬。

## 總結

明朝宗室制度初衷為「親親之誼」，卻因缺乏社會產出與失控的補貼機制，使皇族淪為龐大的寄生體系。這群凌駕於法律之上的「特權階級」最終與明朝政權一同在民怨的火海中覆滅。
