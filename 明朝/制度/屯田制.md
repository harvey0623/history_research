# 屯田制

## 歷史背景

1368年（明洪武元年），明朝建立初期，經歷元末長期戰亂，全國面臨人口銳減、農田荒蕪及國家財政極度困難等嚴峻考驗。為重建經濟秩序並鞏固邊防，明太祖[朱元璋](../皇帝/朱元璋.md)大力推行屯田制度。此制度旨在實行「自給自足」的屯墾政策，達成以下多重戰略目標：

- **解決軍糧供給**：減輕朝廷遠途運糧的龐大財政負擔。
- **恢復農業生產**：招募流民與獎勵開墾，促進農村經濟復甦。
- **開墾邊遠荒地**：增加全國實際耕地面積。
- **降低國家財政負擔**：達到「立國不增加農民賦稅，養兵不耗費國家府庫」的理想狀態。
- **穩定邊疆與社會秩序**：透過人口移徙與邊軍駐紮，鞏固邊境防務與基層治安。

## 制度架構與核心內容

明代屯田制度土地來源多樣，主要包含戰亂後無主荒地（最主要來源）、抄沒官員與豪強之土地、原有官田與公田、政府組織移民重新分配之土地，以及邊疆新開墾之地。屯田體系依執行對象與運作模式，主要分為**軍屯**、**民屯**與**商屯**三大支柱：

### 1. 三種屯田制度比較

| 比較項目     | 軍屯                       | 民屯                       | 商屯                       |
| :----------- | :------------------------- | :------------------------- | :------------------------- |
| **執行者**   | 軍戶                       | 農民、移民                 | 商人                       |
| **制度目的** | 養軍備戰                   | 恢復農業生產與安頓流民     | 補給邊防軍糧               |
| **運作方式** | 士兵平時耕作自給並繳納軍糧 | 招集民戶開墾荒地並按額納稅 | 商人運送糧食至邊境換取鹽引 |
| **國家角色** | 朝廷與衛所直接控制         | 政府行政引導與戶籍管理     | 利用市場機制與商業特許     |
| **主要優點** | 大幅節省國家軍費           | 增加耕地面積、安定社會秩序 | 降低政府遠途物流運輸成本   |
| **主要缺點** | 戰力逐漸下降、軍戶大量逃亡 | 易遭豪強兼併、稅負逐漸加重 | 容易引發官商勾結與制度腐敗 |

### 2. 三種屯田的型態本質

- **軍屯**：平時種田自給，戰時應徵作戰。屬於 military 體制的核心，深度依附於[衛所制](./軍事/衛所制.md)與[戶籍制度](./戶籍制度.md)中的軍戶體系。
- **民屯**：政府鼓勵並組織百姓開墾荒地，屬於農業發展與移民墾殖政策，以恢復社會生產力為主要核心。
- **商屯**：由商人籌集並運送糧食至邊疆衛所，換取政府發放之食鹽專賣憑證（鹽引）。此型態結合了市場機制，與[開中法](./開中法.md)構成密不可分的特許軍需供給體系。

## 歷史影響與崩潰原因

### 1. 歷史影響

屯田制在明朝前期發揮了極大的歷史作用，不僅成功奠定了明初衛所軍制的經濟基礎，大幅降低了邊防養兵成本，更使北方與邊疆廣大荒地得到開發，推動了山西、徽州等地域性商幫的崛起（如商屯推動了邊商資本累積）。

### 2. 各屯田型態之崩壞原因

- **軍屯崩壞**：軍戶世襲制度過於僵化，加上軍田被衛所官員與地方豪強大量侵吞，軍丁淪為私役、生活困頓而引發集體逃亡；同時農兵合一使訓練廢弛，軍隊戰力嚴重衰退。
- **民屯崩壞**：隨著土地兼併日益加劇，小農失去土地；加上政府賦稅與徭役不斷加重，遭遇天災時政府救災體系失靈，民戶被迫流亡。
- **商屯崩壞**：晚明鹽引濫發且遭投機炒作，官商勾結弊端叢生；商人轉為直接繳納白銀（折色），不再於邊境實際開墾與運糧，導致邊軍陷入嚴重缺糧危機。

### 3. 制度性共同崩壞因素

1. **制度僵化**：統治機制無法適應明中後期人口增長與商業貨幣經濟的急速變化。
2. **官僚腐敗與土地兼併**：特權階級侵佔屯田，破壞了原本的土地分配與稅賦基礎。
3. **國家財政惡化**：實物稅收體系解體，中央財政調配能力嚴重不足。
4. **天災戰亂與人口流動**：小農與軍戶大量失業流亡，破壞了基層社會穩定。

### 4. 對明朝滅亡的深遠影響

明末屯田體系的全面失效，直接引發了三重連鎖危機：軍屯崩潰導致國家衛所軍隊失去戰鬥力；商屯崩潰導致邊防軍糧徹底斷絕；民屯崩潰則致使大量農民失地流亡，民變爆發（如李自成起義）。三者同時失效使明朝陷入軍事、財政與社會的三重絕境，最終在內有農民起義、外有後金（清）軍事壓力的雙重夾擊下走向滅亡。

### 5. 與清朝制度之對比

| 比較面向     | 明朝體制                     | 清朝體制                     |
| :----------- | :--------------------------- | :--------------------------- |
| **軍事體制** | 軍屯制（農兵合一、世襲衛所） | 八旗與綠營制（職業軍人體制） |
| **軍糧補給** | 商屯與軍屯自給               | 國家財政統一撥款採購         |
| **農業政策** | 民屯招墾、實物賦稅           | 小農經濟、輕徭薄賦           |
| **財政結構** | 實物納稅與半市場化           | 白銀貨幣化與統一銀兩制       |

- **核心差異**：明朝傾向以「制度直接養軍、養國」，強制綁定軍戶與土地；清朝則轉向以「財政與貨幣稅制」來運作國家機器與維繫專業軍隊。

## 研究結論

明代屯田制度是明初統治者在資源匱乏環境下，成功結合軍事、農業與商業經營的制度創新。它在明前期創造了穩固的經濟與軍事基石。然而，由於過度依賴強制性的戶籍世襲與實物供給，當明中後期社會邁向商品貨幣經濟與土地高度兼併時，屯田體系無法完成制度轉型，最終走向全面崩潰。屯田制的失效不僅反映了明代基層治理體系的解體，亦是導致明朝終局敗亡的核心結構性要因之一。
