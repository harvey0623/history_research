# 明英宗 朱祁鎮（正統 / 天順）

## 歷史背景

明英宗朱祁鎮是明宣宗朱瞻基的長子。`1435年（明宣德十年）`，宣宗暴崩，年僅9歲的朱祁鎮繼位，成為明朝第五位皇帝，年號正統。

正統初年，由於皇帝年幼，朝政由太皇太后張氏主持，並由內閣重臣「內閣三楊」（[楊士奇、楊榮、楊溥](../人物/文臣/內閣三楊.md)）共同輔政。在此期間，朝廷政策延續了「[仁宣之治](../仁宣之治.md)」的清明風氣，邊防穩固，國家依然處於鼎盛時期。然而，隨著張氏與三楊的相繼離世，英宗開始親政，政治格局發生了重大轉變。

## 核心生平與事蹟

1. **宦官專權與盲目北征**
   - **寵信王振**：英宗親政後，極度寵信其貼身宦官王振，甚至尊稱其為「先生」。王振藉此大權獨攬，排斥異己，開啟了明代宦官專權的惡劣先例。
   - **[土木堡之變](../重大事件/英宗危機與復辟/土木堡之變.md)**：`1449年（明正統十四年）`，北方蒙古瓦剌部太師也先大舉南侵。英宗在王振的慫恿與鼓動下，渴望效法祖輩的武功，不顧朝臣的強烈反對，率領二十餘萬京兵精銳倉促北征。由於後勤失調、指揮混亂，明軍在土木堡遭遇瓦剌軍伏擊，全軍覆沒，王振死於亂軍之中，英宗本人則被俘，史稱「[土木堡之變](../重大事件/英宗危機與復辟/土木堡之變.md)」。

2. **[北京保衛戰](../重大事件/英宗危機與復辟/北京保衛戰.md)與南宮幽禁**
   - **移交皇權**：英宗被俘後，為了防止瓦剌以皇帝要挾朝廷，兵部尚書于謙等文官集團擁立英宗之弟郕王朱祁鈺為帝（即景泰帝），尊英宗為太上皇，並成功組織了[北京保衛戰](../重大事件/英宗危機與復辟/北京保衛戰.md)，擊退瓦剌。
   - **七年幽禁**：`1450年（明景泰元年）`，瓦剌見英宗已無勒索價值，將其放回南京。景泰帝因擔心皇位不保，將英宗幽禁於南宮，並對其進行嚴密的監視與防範，時間長達七年。

3. **[奪門之變](../重大事件/英宗危機與復辟/奪門之變.md)與復辟**
   - **重登帝位**：`1457年（明景泰八年）`，景泰帝重病且無子嗣，朝政陷入權力真空。將領石亨、政客徐有貞、宦官曹吉祥等人聯手發動政變，迎接英宗重登大寶，史稱「[奪門之變](../重大事件/英宗危機與復辟/奪門之變.md)」。英宗復位後改元天順。

4. **復辟後的統治與政治汙點**
   - **冤殺于謙**：英宗復辟後，為了確立自身政權的合法性，聽信徐有貞等人的讒言，以「迎立外藩」的莫須有罪名將拯救大明的功臣于謙逮捕並處死。這成為他一生中最大的政治汙點。

## 歷史影響與後世評價

- **明朝國勢的轉折點**：由英宗主導的土木堡之變，使得明朝建國以來的京軍精銳（如三大營）折損殆盡，大批文武大臣殉國。此戰徹底扭轉了明朝對北方遊牧民族的戰略攻勢，被迫轉為全面戰略防禦，國勢從此走下坡。
- **廢除[殉葬制度](../制度/殉葬制度.md)**：雖然英宗在政治與軍事上昏庸平庸，但他晚年做出一項極具人道主義色彩的決策。他在臨終前下詔，廢除了自明太祖朱元璋以來重新啟用的殘酷[殉葬制度](../制度/殉葬制度.md)，使得後代嬪妃免於被迫殉死。這一德政為他贏得了後世的讚譽。

## 研究結論

明英宗朱祁鎮是明代歷史上經歷最為奇特、起伏最大的君主。他一生兩度登基，做過俘虜也當過太上皇，被幽禁七年後又成功復辟。作為一個皇帝，他在政治上平庸無能，聽信小人，且有冤殺功臣于謙的大過；但作為一個人，他性格溫和，並在臨終前廢除殘酷的殉葬制度，展現了人性的慈悲，使得他成為歷史上一個極具爭議與複雜性的人物。
