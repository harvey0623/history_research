# 明憲宗 朱見深 (成化)

明憲宗朱見深，年號成化（1465年－1487年），是明朝歷史上一位極具爭議性但又展現出高超守成手腕的皇帝。他的一生從充滿恐懼的幼年開始，歷經政局動盪，最終開創了「成化中興」。

---

## 一、 幼年創傷：皇權更迭下的陰影

朱見深的性格與治國風格深受其早年不幸經歷影響：

- **政治變局**：1449年（正統十四年），其父英宗在[土木堡之變](../重大事件/英宗危機與復辟/土木堡之變.md)被俘，兩歲的朱見深被立為皇太子。然而，1452年（景泰三年），其叔景泰帝朱祁鈺為傳位親子，將五歲的朱見深廢為沂王，並將其趕出東宮軟禁。
- **身心重創**：在長達五年的幽禁歲月中，他身邊充斥耳目，看盡人情冷暖。這種隨時面臨死亡威脅的高壓環境，導致他變得極度自卑，並患有嚴重口吃。
- **治國影響**：因害怕臨場問答時口吃發作引來尷尬，朱見深登基後極度怕生，成為明朝第一位**不願意單獨召見大臣議事**的皇帝，主要透過批閱奏摺或內閣代傳處理政務。

## 二、 情感歸宿：萬貴妃與「專寵」真相

朱見深對大他17歲的萬貴妃（萬貞兒）有著超越常理的依戀，這段感情是理解成化朝政局的關鍵：

### 1. 心理依託：從「守護者」到「精神支柱」

- **生死相依的童年**：在朱見深5歲被廢、軟禁宮外的最黑暗時期，萬貞兒是唯一對他不離不棄的宮女。她兼具了保姆、母親與守護者的角色，在隨時可能被暗殺的高壓環境下，萬貞兒是朱見深唯一的安全感來源。
- **不可取代的依戀**：朱見深曾對周太后直言：「只有在萬貞兒的撫摸下才能安然入睡」。這種深刻的心理補償，使得萬氏即便年老色衰，在朱見深心中依然是至高無上的存在。

### 2. 權力運作：後宮與外廷的結合

- **廢后風波與後宮權威**：成化元年，第一任皇后吳氏因杖責萬氏，僅入宮一個月就被朱見深廢黜。此舉確立了萬氏「實質后宮首領」的地位，後來的王皇后只能以「萬事不管」來保全位子。
- **傳奉官與利益共同體**：萬貴妃透過與宦官（如汪直、梁芳）及外廷官員勾結，發展出「**傳奉官**」制度（不經吏部，由皇帝直接任命）。這使昭德宮（萬氏居所）成為權力轉運站，形成了一個龐大的政治利益集團。

### 3. 後宮威懾：強勢的統治風格

萬貴妃在後宮展現了極其強悍且不容挑戰的威懾力，這使得即便貴為皇后的女性也對其避讓三分：

- **無視禮制與挑戰皇后**：面對第一任吳皇后，萬氏處處搶風頭，引發吳皇后杖責卻最終導致吳氏被廢。面對繼任的王皇后，萬氏依然傲慢，在重要慶典朝見時甚至拒不向皇后行禮。
- **車駕僭越與出巡威儀**：每當前往西苑等地遊玩時，萬貴妃的車駕常直接走在王皇后之前。王皇后為求自保，對此種嚴重僭越禮制的行為只能默許與忍受。
- **身穿戎裝的心理施壓**：萬貴妃性格豪爽且帶有英氣，她時常一身戎裝陪伴皇帝出遊。甚至在皇帝準備臨幸其他嬪妃時，她會刻意穿上軍裝、騎馬走在隊伍最前方，以此向承寵者武力示威並施加心理壓力。

### 4. 爭議真相：她真的殺害皇子嗎？

萬貴妃「毒殺皇子、強迫墮胎」的惡名，主要源於清修《明史》，但歷史研究顯示這極大機率是刻意的政治抹黑。

- **謠言的來源與孤證**：
  - **非當代記載**：成化當代的官方記錄（如《明實錄》）完全沒有萬氏殺害皇子的記錄。
  - **百餘年後的傳聞**：相關指控最早見於明末萬曆年間的筆記《穀山筆麈》，據傳是源自老太監的口耳相傳。這些傳聞隨後被清初文人毛奇齡與《明史》編纂者採納，缺乏同時期的物證或文書支持。
- **「隱藏皇子」傳說的邏輯漏洞**：
  - **朱祐樘（弘治帝）傳奇**：民間流傳紀氏在宮中暗中撫養皇子六年而萬氏不知情。但在制度森嚴、耳目眾多的明朝後宮，要在西廠與萬氏權力網下隱瞞一名皇子六年，實屬天方夜譚。
  - **真實情境**：朱見深對紀氏懷孕應是知情的，並將其移往安樂堂保護。萬氏得知皇子出生後亦曾送禮，並無傳聞中的瘋狂迫害。
- **數據勝於雄辯：豐沛的子嗣**：
  - 朱見深一生育有**14個兒子與6個女兒**。這些子嗣多數出生於萬貴妃權勢最巔峰的成化十幾年。若萬氏真有「滅絕皇嗣」的政策，大明皇室理應面臨絕嗣，而非繁衍興旺。
- **乾隆皇帝的「官方闢謠」**：
  - 清朝乾隆皇帝在閱覽《明史》後，曾親自撰文《駁明史萬妃傳》。他以皇帝的視角指出，若萬氏真能如此隻手遮天，朱見深不可能坐視自己絕後。乾隆認為這些情節「太過荒唐，完全不合邏輯」，並斥之為荒謬的八卦。

### 5. 抹黑動機：成化犁庭的政治代價

- **清朝的報復性修史**：朱見深發動「**成化犁庭**」重創了清朝先祖（建州女真），斬殺其五世祖董山。這份深仇大恨促使清朝在修撰《明史》時，必須將朱見深塑造成「任由妖妃與宦官擺布」的昏君，以削弱其統治的合法性並掩蓋其軍事成就。

### 6. 淒美結局：生同衾，死同穴

- 1487年（成化二十三年），萬貴妃去世，朱見深悲慟感嘆：「貞兒不在，我亦不久矣。」幾個月後，朱見深果然因思念過度、憂鬱成疾而駕崩。他更打破慣例，將萬氏葬在天壽山帝陵區，誓言死後亦要相守。

## 三、 成化中興：化解五大帝國危機

1464年（天順八年），17歲的朱見深繼位，當時的大明滿目瘡痍，他展現了傑出的政治手腕：

### 1. 凝聚朝堂向心力

- **平反冤案**：立刻為[于謙](../重大事件/英宗危機與復辟/北京保衛戰.md)平反，並恢復景泰帝帝號，撫慰人心，化解文官集團裂痕。
- **清除權臣**：果斷處置「奪門之變」的投機份子，並重用李賢、商輅等賢臣，扭轉腐敗吏治。

### 2. 鐵血應對外患：成化犁庭

- **背景**：東北建州女真（首領董山）採取「兩面手法」，一邊朝貢領賞，一邊劫掠遼東。
- **軍事打擊**：朱見深決定採取毀滅性的「**犁庭掃穴**」策略。先扣押董山，隨後派軍隊聯合朝鮮直搗黃龍。
- **成果**：西廠提督**汪直**親自監軍執行，將建州女真徹底打殘，為明朝東北邊境換來137年的安寧。

### 3. 疏導取代鎮壓：解決流民危機

成化年間，大明面臨了嚴峻的流民問題，僅荊襄山區（鄂、豫、陝交界）就聚集了高達上百萬無家可歸的人口。

**流民大量產生的四大核心因素：**

- **極端氣候與頻發天災**：受**小冰期**影響，北方（如山東）頻發大旱與蝗災，南方（如江西、湖廣）則暴雨成災。糧食絕收迫使百姓離鄉背井。
- **土地兼併與賦稅壓力**：嚴重的土地兼併與沉重的賦稅，逼迫失去土地的農民逃入官府戶籍難以管轄的山區以求生存。
- **宦官橫徵暴斂與經濟破壞**：鎮守太監利用特權瘋狂盤剝。例如雲南太監錢能坑騙大商人，導致大量受僱夥計失業並淪為流民，破壞了地方經濟基石。
- **連年戰亂與社會動盪**：自「土木堡之變」後，社會秩序受創，失去活路的底層百姓紛紛逃往地形複雜的深山。

**解決方案：鄖陽模式**

- **設立鄖陽府**：面對隨時可能叛亂的百萬流民，朱見深採納官員建議，不採武力屠殺，而是設立專門管理流民的「**鄖陽府**」。
- **合法化與生產力轉化**：允許流民就地登記戶籍、分發土地並減免賦稅，成功將破壞力量轉化為帝國的生產力，徹底化解了足以動搖國本的危機。

### 4. 平定內亂與邊防

- **內亂**：派韓雍平定兩廣壯瑤叛亂（設立兩廣總督），派李瑾平定四川蠻苗暴動。
- **外防**：對蒙古韃靼取得「威寧海大捷」，成功收復河套地區。

### 5. 務實改革復甦經濟

- **財政改革**：推廣「折銀徵收」（適應白銀流通趨勢），改革鹽法。
- **開放貿易**：在廣東、福建設立市舶司，推動「農商並重」，造就江南絲織業與景德鎮瓷器業的繁榮。

## 四、 統治特性：傑出的守成之君

朱見深被歷史定位為**「守成型君主」**：

- **治國風格**：冷靜的現實主義，務實且懂得分寸。
- **垂拱而治**：他看似懦弱散漫，不與大臣爭執，但在平反冤案、安置流民、成化犁庭等關鍵決斷上極其精準。他成功替大明王朝拆除了致命的「未爆彈」，穩住了帝國陣腳。

## 五、 晚年弊政：體制僵化與怠政

執政後期，朱見深因怠政與怕生，導致權力結構發生畸變：

- **「紙糊」與「泥塑」**：
  - **紙糊三閣老**（萬安、劉珝、劉吉）：依附萬貴妃，毫無作為，喪失文臣氣節。
  - **泥塑六尚書**：六部高官不作為，導致國家僅靠慣性運轉。
- **宦官專權與特務政治**：皇帝退居幕後，使信任向內廷傾斜。
  - **汪直與西廠的興衰**：汪直身為西廠提督，曾是朱見深最重要的軍事與特務利器，但其最終因權力傾軌而失寵。
    - **被疏遠的原因**：
      1. **長期離京**：因頻繁被派往邊疆（如大同）監軍與征戰，與皇帝的感情依賴隨時間變淡。
      2. **權力鬥爭與「演戲圈套」**：汪直的囂張引發其他宦官不滿。政敵安排小太監演戲，表現出「不怕皇帝只怕汪太監」的假象，引發朱見深的猜忌。
      3. **東廠背叛**：東廠總管尚銘暗中揭發汪直惡行，成為壓垮信任的最後一根稻草。
      4. **失去戰略價值**：在「成化犁庭」等任務完成後，朱見深認為制衡文官與清除邊患的目標已達成，遂將其邊緣化。
    - **汪直的結局**：與明朝其他權監（如魏忠賢）的慘死不同，汪直得以**保全性命並善終**。成化十八年西廠撤銷，汪直先後被貶至南京御馬監及最底層職位「奉御」（負責草料）。朱見深展現了寬厚有情的一面，並未殺他或抄家，讓他以底層太監身分平靜走完餘生。
- **沉迷方術**：後期沉溺於修仙，法紀鬆弛...

---

**總結**：朱見深在「土木堡」後的廢墟上，以守成之姿穩住了大明江山。他雖有晚年怠政之失，但其軍事與內政成就（成化中興）在明代歷史中具有不可磨滅的地位。
