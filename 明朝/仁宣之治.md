# 仁宣之治

明朝在「仁宣之治」（明仁宗與明宣宗統治時期）透過一系列全方位的休養生息政策，成功恢復了因永樂時期連年征戰與大型工程而耗散的國力。具體作法可分為以下幾個方面：

- **軍事與外交上的戰略收縮**：
  - **自安南（交趾）撤兵**：永樂時期強行佔領安南，引發當地人強烈反抗，雙方反覆交戰20多年，無數人力與財力被消耗在此泥淖中。明宣宗一上台便決定從安南撤兵，果斷放下了這個沉重的財政與軍事包袱。
  - **對蒙古改採「懷柔政策」**：有別於明成祖朱棣耗費巨大的御駕親征，仁宣時期選擇以經濟手段換取和平。朝廷放寬對蒙古部落的朝貢政策，給予首領封號，並允許他們以馬匹交換中原的糧食、茶葉和布匹。這種以貿易和賞賜取代戰爭的做法，不僅大幅節省了軍費開支，也成功保障了邊境的安寧。

- **經濟與民生上的減負與振興**：
  - **叫停「面子工程」**：仁宗即位後，立刻停止了如「鄭和下西洋」這類勞民傷財且極度燒錢的活動。即使宣宗時期曾有過一次復行，其性質也轉變為較為節儉的外交往來。
  - **全面赦免民間欠稅**：永樂時期許多百姓因負擔沉重而累積大量欠稅，甚至因此流亡成為流民。仁宗將這些欠糧全面赦免，徹底卸下了百姓的重擔，讓流民得以安心回到自己的土地上重新展開農耕生活，這項賦稅改革極大地促進了農業生產的恢復。

- **政治環境的寬鬆與撥亂反正**：
  - **平反冤獄與緩和政治氛圍**：仁宗即位後，替永樂時期因朱棣篡位而被殺戮或監禁的建文帝舊臣平反，並釋放獄中大臣，一改過去恐怖的政治氛圍。
  - **重用賢臣與確立內閣制度**：任用被後世稱頌的賢臣「三楊」（楊士奇、楊榮、楊溥）組成內閣，確立了「票擬制度」，讓官僚系統得以順暢、高效地運作，為國家的休養生息提供了穩定的政治後盾。

透過這些減少國家擴張性消耗、減輕百姓負擔並重塑安定政治環境的政策，明朝民間社會展現了驚人的恢復力。短時間內，民間便重新變得富裕，原本空虛的國庫也再次充盈，成就了被後人視為明朝政治最清明的「仁宣之治」。
